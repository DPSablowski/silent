# silent

SILENT - Spectrograph sImuLation and EmulatioN Tool

Written in C++ with Qt in Qt Creator; all files in folder SILENT.

You can type

objdump -p Silent | grep NEEDED

to show all necessary dependencies:

libQt5PrintSupport

libQt5Widgets

libQt5Gui

libQt5Core

libGL

libpthread

libstdc++

libm

libgcc_s

libc

to run the program: ./Silent
